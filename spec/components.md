# コンポーネント詳細

CLAUDE.md のコンポーネント表の詳細版（SoT）。CLAUDE.md には 1 行サマリのみを置き、
設計経緯・根拠・issue/ADR 参照を含む詳細はこのファイルに記録する。

**運用ルール**: 新コンポーネント追加・既存変更時は、このファイルに詳細を書き、
CLAUDE.md のサマリ表には 1 行（名前 + 一言 + 参照）だけ追記する。
