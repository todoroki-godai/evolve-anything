# コミット時のバージョン管理（PJ固有）
bump 時は `.claude-plugin/plugin.json` + `.claude-plugin/marketplace.json` plugins[0].version + CHANGELOG.md を同期更新する。main マージ後に `claude plugin tag --push` で `evolve-anything--v<version>` タグを作成。その後 `bin/evolve-release-sync` を実行してローカル開発環境のプラグインを最新版へ同期する（marketplace は Directory source = ローカル作業ディレクトリを見るため、ローカル main を pull しないと `claude plugin update` が古いバージョンを返す）。最後に `/evolve-anything:docs-refresh` を実行して `docs/site/` を最新化する。
