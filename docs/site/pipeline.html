パイプライン

/rl-anything:evolve を実行すると、以下の5ステージが順番に走ります。各ステージは独立して呼び出すことも可能です。

👁

Observe

15個の hooks が LLM コストゼロでセッションの使用状況・エラー・修正フィードバック・ファイル変更を自動記録する。データは sessions.jsonl・corrections.jsonl・usage.jsonl に蓄積される。

🔍

Diagnose

蓄積データを分析して低スコアスキルを特定。skill_triage が CREATE / UPDATE / SPLIT / MERGE / OK の5択判定を下す。trigger_eval_generator が evals.json を自動生成する。

⚙️

Compile

genetic-prompt-optimizer が LLM 1パスでパッチを生成・適用する。regression_gate.py が品質低下を検知してロールバックする。

🧹

Housekeeping

古い corrections エントリの整理、マージ済みスキルの削除候補提示、ファイルサイズバジェット超過の警告を行う。

📋

Report

audit レポートを生成。growth_level.py が env_score (0.0–1.0) を Lv.1–10 のタイトルにマッピングして表示する。

✅ Auto Trigger — セッション終了・corrections 蓄積・ファイル変更時に evolve / audit の実行を自動提案する。userConfig でトリガー閾値・クールダウンを調整できる。

主要スキル

rl-anything が提供するスキルの一覧です。すべて /rl-anything:<skill> で呼び出せます。

採点・進化系

rl-scorer オーケストレーター (Haiku) + 3サブエージェント並列 (tech/struct=Haiku, domain=Sonnet) で3軸採点。CLAUDE.md からドメイン（ゲーム / API / Bot / ドキュメント）を自動判定して評価軸を切り替える。
skill-triage テレメトリ + trigger eval で CREATE / UPDATE / SPLIT / MERGE / OK の5択判定を行う。scripts/lib/skill_triage.py 実装。
evolve-skill 特定スキルに自己進化パターン（Pre-flight / pitfalls.md）をピンポイントで組み込む。適性判定 → テンプレート組み込み → 人間確認を1コマンドで実行。
evolve-fitness accept / reject データを分析してスコアと採用率の相関問題を発見し、適応度関数の改善を提案する。変更は必ず人間の承認が必要。
generate-fitness CLAUDE.md・rules・skills からドメイン特性を分析し、プロジェクト固有の適応度関数を自動生成する。

管理・補助系

spec-keeper SPEC.md と ADR（Architecture Decision Record）を管理する。機能完了時に仕様を最新化し、設計判断を ADR として記録する。init / update / recover の3モード。
second-opinion Claude Agent による cold-read セカンドオピニオン。codex 不要で Agent ツールのみで動作する。
implement plan artifact から複雑性アセスメント → shallow / standard / deep の深度別実行 → テレメトリ記録まで一貫して処理する。depends_on グラフによる実行順序制御と循環依存チェックをサポート。
agent-brushup ~/.claude/agents/ のエージェント定義を品質診断し、改善提案・新規作成・削除候補を提示する。upstream 更新チェック付き。
cleanup PR マージ・デプロイ後の後片付け。ブランチ削除・worktree 削除・Issue close 候補を候補提示→個別承認→実行で安全に実行する。
breakthrough 「惜しいがブレイクスルーしない」問題を診断→戦略提案→Agent 起動で突破する。
reflect corrections を対話的にレビューし CLAUDE.md / rules / auto-memory に反映する。ユーザー修正を適切なメモリ層にルーティングする。
import コミュニティスキルを owner/repo・ローカルパス・URL からインポートしてインストールする。パストラバーサル多層防御付き。
docs-refresh docs/site/ の HTML 説明サイトをリポジトリの現状（バージョン番号・スキル一覧・4つの柱・アーキテクチャ表）に自動更新する。
release-notes-review CC リリースノートをプロジェクト環境・グローバル環境と突合し、適用可能な新機能を優先度別に報告。環境健康診断も実施する。
pitfall-curate 任意PJの pitfalls.md を育てる PJ非依存スキル。重複排除・普遍性分類・配布版生成・同期ゲートに加え、自動強制 hook の有効化も skill 1発で行う（pitfalls.md を自動発見し確認のうえ登録）。
